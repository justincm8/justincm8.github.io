    • Interface vs. interface

  • An interface is a general programming concept and an interface is a specific programming construct in Java.
  • Generally an interface is just a way of interacting with some code.
  • A class's interface is its public methods.
  • The interface of the Course class includes the constructor, getDepartment, setNumber.

    • interface

  • An interface lets us create a contract; a guarantee that a class with implement certain methods.
  • All methods declared in an interface are abstract and public.

    • interface

  • When we create a class that implements an interface, it must override all of the interfaces methods.
