Variables

Three things are happening here:

  1. We are telling the computer that we are going to store data in a variable named num.
  2. We tell the computer that the variable num will store integer data (int).
  3. The initial value of num is set to zero.

Declaration and Initialization

  • Declaration: Telling the computer to reserve some space to store some data an associate it with a label.
  • Initialization: Setting the initial value stored by the variable.
  • It's generally considered bad practice to do these as separate steps.

Types

Data in Java is represented by combinations of 8 different types known as the Java primitive types. We will use the following four in this course. You can read about the others here.
  • int: represents integer values.
  • double: represents floating point (decimal) values.
  • boolean: represents whether something is true or false.
  • char: represents a single character.

int

  • Represents an integer, a number wihtout a decimal component.
  • Can store values from -2,147,483,648 to 2,147,483,647.
  1. Show short and long.

double

  • Stores a floating point value (numbers with decimal values).

boolean

  • Can only have one of two values: true or false.

char

  • Represents a single character value.
  • Must be encolsed in single quotes not double.

Literals

            
                int i = 10; // 10 is a literal, i is a variable
                double threshold = 0.5; // 0.5 is a literal, threshold is a variable
                boolean isACat = true; // true is a literal, isACat is a variable
                char favorite = '8'; // 8 is a literal, favorite is a variable
  • A literal is a value that appears directly in your code.
